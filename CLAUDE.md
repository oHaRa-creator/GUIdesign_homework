## Project status

This repository is currently a near-empty scaffold. It contains only `README.md` and
`.gitattributes` — there is **no source code, build tooling, dependency manifest, or tests yet**.

The project is a GUI design homework assignment. Per the README, the goal is to build a
"役に立たないGUI" (a deliberately *useless* / pointless GUI — i.e. the "useless machine" genre
of UI, where the interaction itself is the point rather than any practical function).
